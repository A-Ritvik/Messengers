# Messengers
<img width="1109" height="505" alt="Screenshot 2026-07-01 121241" src="https://github.com/user-attachments/assets/3363a718-4e06-4fd0-ab7e-653130f003a4" />

This is a 2d game where you play as a knight who delivers mail between a king (in his castle) and a dark wizard in the caves. Your goal is to one day reach the horizon (the tickets a 100 coins). Bandits also exist out in the open wild, who have one coin each. Delivering mail also gets you some coins. The Merchant sells the ticket to the horizon along with speed, health, and attack potions. 

# Controls
The controls are:
A/D movement; W/A/S/D for movement when inside the castle
F- Deliver/Pickup mail
S- attack
E- enter shop
Esc- pause/leave shop
R - enter ladder
Space - Jump

# UI 
Features an overlay UI with a heart that gradually darkens as the player gets closer to death, the coin display, and a button to access the pause menu. The Pause menu features a back to start menu button (progress is not saved for now). There is also a game over screen on player death.
